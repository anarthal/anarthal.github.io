WritableField and WritableFieldTuple concepts

A type is a WritableField if it can be used to represent a statement parameter value to be sent to the server. A type T satisfies WritableField if it is any of the types listed in this table. More types may be added in future releases.

A type T is a WritableFieldTuple if it's a std::tuple specialization, or a reference to one, and all element types fulfill WritableField, or are (possibly cv-qualified) references to WritableField types. Empty tuples satisfy WritableFieldTuple.

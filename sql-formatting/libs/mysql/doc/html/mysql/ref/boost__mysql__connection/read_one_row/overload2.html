connection::read_one_row (2 of 2 overloads)

Reads a single row.

Synopsis 
row_view
read_one_row(
    execution_state& st);
Description

If a row was read successfully, returns a non-empty row_view. If there were no more rows to read, returns an empty row_view.

The returned view points into memory owned by *this. It will be valid until the underlying stream performs any other read operation or is destroyed.

stmust have previously been populated by a function starting the multifunction operation, like start_query or start_statement_execution. Otherwise, the results are undefined.

This operation involves only reads on the underlying stream.

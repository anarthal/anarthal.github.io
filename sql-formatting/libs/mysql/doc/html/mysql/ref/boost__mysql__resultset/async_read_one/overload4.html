resultset::async_read_one (4 of 4 overloads)

Reads a single row as a row_view.

Synopsis 
template<
    class CompletionToken>
auto
async_read_one(
    use_views_t,
    error_info& output_info,
    CompletionToken&& token);
Description

If a row was read successfully, returns a non-empty row_view. If there were no more rows to read, returns an empty row_view.

The returned view points into the connection that *this references. It will be valid until the connection performs any other read operation or is destroyed. The handler signature for this operation is void(boost::mysql::error_code, boost::mysql::row_view).

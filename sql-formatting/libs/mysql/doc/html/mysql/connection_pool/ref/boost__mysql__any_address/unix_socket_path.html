any_address::unix_socket_path

Retrieves the UNIX socket path that this object contains.

Synopsis 
string_view
unix_socket_path() const;
Preconditions

this->type() == address_type::unix_path

Object lifetimes

The returned view points into *this, and is valid as long as *this is alive and hasn't been assigned to or moved from.

Exception safety

No-throw guarantee.

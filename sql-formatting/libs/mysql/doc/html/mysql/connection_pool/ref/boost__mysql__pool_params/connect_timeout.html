pool_params::connect_timeout

The timeout to use when connecting.

Synopsis 
std::chrono::steady_clock::duration connect_timeout {std::chrono::seconds(20)};
Description

Connections will be connected by the pool before being handed to the user (as per any_connection::async_connect). If the operation takes longer than this timeout, the operation will be interrupted and be considered as failed.

Set this timeout to zero to disable it.

This value must not be negative.

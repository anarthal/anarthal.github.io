field_view::get_blob

Retrieves the underlying value as a blob (unchecked access).

Synopsis 
constexpr blob_view
get_blob() const;
Preconditions

this->is_blob() == true (if violated, results in undefined behavior).

Exception safety

No-throw guarantee.

Object lifetimes

The returned view has the same lifetime rules as *this (it's valid as long as *this is valid).

resultset_base::info

Additionat text information about the execution of the SQL statement that generated this resultset.

Synopsis 
boost::string_view
info() const;
Description

The resultset must be complete before calling this function.

This function returns a view object, with reference semantics. This view object references *this internal state, and will be valid as long as *this (or a resultset move-constructed from *this) is alive.

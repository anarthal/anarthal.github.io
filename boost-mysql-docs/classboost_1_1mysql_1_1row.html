boost::mysql::row Class Reference
Resultsets

Description

Represents a row returned from a query.

Call row::values() to get the actual sequence of mysql::value. There will be the same number of values and in the same order as fields in the SQL query that produced the row. You can get more information about these fields using resultset::fields().

If any of the values is a string (mysql::value having string_view as actual type), it will point to an externally owned piece of memory. Thus, the row base class is not owning; this is contrary to owning_row, that actually owns the string memory of its values.

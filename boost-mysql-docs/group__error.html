MySQL-specific error codes.

Some error codes are defined by the client library, and others are returned from the server. For the latter, the numeric value and string descriptions match the ones described in the MySQL documentation. Only the first ones are documented here. For the latter, see https://dev.mysql.com/doc/refman/8.0/en/server-error-reference.html

Enumerator
ok 

No error.

Returned as the result of a text query or a statement execution. It is an I/O object that allows reading rows progressively. This section provides an in-depth explanation of the mechanics of this class. Resultsets are default-constructible. A default-constructed resultset has resultset::valid return false. Calling any member function on an invalid resultset, other than assignment, results in undefined behavior. Resultsets are movable but not copyable.

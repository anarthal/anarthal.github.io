connection::async_query (1 of 2 overloads)

Executes a SQL text query (async without error_info version).

Synopsis 
template<
    class CompletionToken>
auto
async_query(
    boost::string_view query_string,
    CompletionToken&& token);
Description

See this section for more info. After the operation completes, you should read the entire resultset before calling any function that involves communication with the server over this connection. Otherwise, the results are undefined. The handler signature for this operation is void(boost::mysql::error_code, boost::mysql::resultset<Stream>).

ssl_mode

Determines whether to use TLS for the connection or not.

Synopsis

Defined in header <boost/mysql/connection_params.hpp> 

enum ssl_mode
Values

Name

Description

disable

Never use TLS Test.

enable

Use TLS if the server supports it, fall back to non-encrypted connection if it does not. Test.

require

Always use TLS; abort the connection if the server does not support it. Test.

socket_connection::connect (1 of 2 overloads)

Performs a connection to the MySQL server (sync with error code version).

Synopsis 
void
connect(
    const endpoint_type& endpoint,
    const connection_params& params,
    error_code& ec,
    error_info& info);
Description

Connects the underlying socket and then performs the handshake with the server. The underlying socket is closed in case of error. Prefer this function to connection::handshake.

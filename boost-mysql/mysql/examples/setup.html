Setup

All examples assume you have a MySQL server running in localhost. Examples make use of a database named boost_mysql_examples. The username and password to used are passed using the command line.

You can set the required database up by running (this assumes the root password is empty): 

> mysql -u root < PATH/TO/BOOST/MYSQL/example/db_setup.sql

This will also create a user example_user with a password example_password, with the appropriate privileges on the database.

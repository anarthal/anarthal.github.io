(3)

Binds parameters to a statement.

Creates an object that packages *this and the statement actual parameters params. This object can be passed to execute, start_execution and their async counterparts.

The parameters are copied into a std::tuple by using std::make_tuple. This function only participates in overload resolution if std::make_tuple(FWD(args)...) yields a WritableFieldTuple. Equivalent to this->bind(std::make_tuple(std::forward(params)...)).

This function doesn’t involve communication with the server.

Preconditions

this->valid() == true

Exception safety

Strong guarantee. Only throws if constructing any of the internal tuple elements throws.

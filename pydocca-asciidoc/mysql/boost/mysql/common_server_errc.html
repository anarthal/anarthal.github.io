boost::mysql::common_server_errc

Defined in header <boost/mysql/common_server_errc.hpp>

enum class common_server_errc: int;

Server-defined error codes, shared between MySQL and MariaDB.

The numeric value and semantics match the ones described in the MySQL documentation. For more info, consult the error reference for [@https://dev.mysql.com/doc/mysql-errors/8.0/en/server-error-reference.html MySQL 8.0], [@https://dev.mysql.com/doc/mysql-errors/5.7/en/server-error-reference.html MySQL 5.7], [@https://mariadb.com/kb/en/mariadb-error-codes/ MariaDB].

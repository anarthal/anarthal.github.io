boost::mysql::field::get_uint64

const std::uint64_t&
get_uint64() const noexcept;

(1)

 
std::uint64_t&
get_uint64() noexcept;

(2)

Retrieves a reference to the underlying std::uint64_t value (unchecked access).

Preconditions

this->is_uint64() == true (if violated, results in undefined behavior).

Exception safety

No-throw guarantee.

Object lifetimes

The returned reference is valid as long as *this is alive and no function that invalidates references is called on *this.

boost::mysql::field::operator=

Replaces *this with v, changing the kind to v.kind() and destroying any previous contents.

Synopsis

Declared in <boost/mysql/field.hpp>

field&
operator=(std::string_view v);

Description

Exception safety

Basic guarantee. Internal allocations may throw.

Object lifetimes

Invalidates references to *this obtained by as_xxx and get_xxx functions, but not the ones obtained by field::operator field_view().

*this is guaranteed to be valid even after v becomes invalid.

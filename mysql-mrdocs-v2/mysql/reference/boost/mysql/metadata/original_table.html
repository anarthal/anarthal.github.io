boost::mysql::metadata::original_table

Returns the name of the physical table the column belongs to.

Synopsis

Declared in <boost/mysql/metadata.hpp>

string_view
original_table() const noexcept;

Description

E.g. in "SELECT id FROM employees emp", original_table() will be "employees".

This is optional information ‐ it won't be populated unless the connection executing the query has meta_mode() == metadata_mode::full.

Exception safety

No‐throw guarantee.

Object lifetimes

The returned reference is valid as long as *this is alive and hasn't been assigned to or moved from.

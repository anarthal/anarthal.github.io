value::value (3 of 3 overloads)

Initialization constructor.

Synopsis 
template<
    class T>
constexpr
value(
    const T& v);
Description

Initializes *this with the same type and value that variant_type(v) would contain. The following exceptions apply:

  • If T is any unsigned integer, the type will be std::uint64_t and the value, std::uint64_t(v).
  • If T is any signed integer, the type will be std::int64_t and the value, std::int64_t(v).

Examples:

  • value(48) yields std::int64_t.
  • value(std::uint8_t(2)) yields std::uint64_t.
  • value("test") yields boost::string_view.

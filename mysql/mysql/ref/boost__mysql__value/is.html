value::is

Checks if the current type alternative is T.

Synopsis 
template<
    class T>
constexpr bool
is() const;
Description

T should be one of value::variant_type's type alternatives. This function does not take into account the possible type conversions. It returns true only if the current alternative matches T exactly. See value::is_convertible_to for a version of this function taking conversions into account. This function is faster than value::is_convertible_to.

value::get_std_optional

Retrieves the stored value as a std::optional.

Synopsis 
template<
    class T>
constexpr std::optional< T >
get_std_optional() const;
Description

T should be one of value::variant_type's type alternatives. If the stored value is a T, or can be converted to T using one of the allowed conversions, returns an optional containing the converted value. Otherwise returns an empty optional.

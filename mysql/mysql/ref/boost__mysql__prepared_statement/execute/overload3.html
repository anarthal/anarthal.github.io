prepared_statement::execute (3 of 4 overloads)

Executes a statement (execute_params, sync with error code version).

Synopsis 
template<
    class ValueForwardIterator>
resultset< Stream >
execute(
    const execute_params< ValueForwardIterator >& params,
    error_code&,
    error_info&);
Description

ValueForwardIterator should meet the ValueForwardIterator requirements. The range [params.first(), params.last()) will be used as parameters for statement execution. They should be a valid iterator range. After this function has returned, you should read the entire resultset before calling any function that involves communication with the server over this connection. Otherwise, the results are undefined.

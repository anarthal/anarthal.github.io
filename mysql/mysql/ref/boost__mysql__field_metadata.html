field_metadata [constructor]

Default constructor.

field_name

Returns the actual name of the field.

has_no_default_value

Returns true if the field does not have a default value.

is_auto_increment

Returns true if the field is defined as AUTO_INCREMENT.

is_multiple_key

Returns true if the field is part of a KEY (but not a UNIQUE KEY or PRIMARY KEY).

is_not_null

Returns true if the field is not allowed to be NULL, false if it is nullable.

is_primary_key

Returns true if the field is part of a PRIMARY KEY.

is_set_to_now_on_update

Returns true if the field is defined as ON UPDATE CURRENT_TIMESTAMP.

is_unique_key

Returns true if the field is part of a UNIQUE KEY (but not a PRIMARY KEY).

is_unsigned

Returns true if the field has no sign (is UNSIGNED).

is_zerofill

Returns true if the field is defined as ZEROFILL (padded to its maximum length by zeros).

original_field_name

Returns the original (physical) name of the field.

original_table

Returns the name of the physical table the field belongs to.

table

Returns the name of the virtual table the field belongs to.

type

Returns the type of the field (see field_type for more info).

Description

All strings point into externally owned memory. The object will be valid while the parent object is alive (typically, a resultset object).

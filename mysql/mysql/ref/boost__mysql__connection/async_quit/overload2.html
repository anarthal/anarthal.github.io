connection::async_quit (2 of 2 overloads)

Notifies the MySQL server that the client wants to end the session (async with error_info version).

Synopsis 
template<
    class CompletionToken>
auto
async_quit(
    error_info& output_info,
    CompletionToken&& token);
Description

Sends a quit request to the MySQL server. If the connection is using SSL, this function will also perform the SSL shutdown. You should close the underlying physical connection after calling this function. If the Stream template parameter fulfills the SocketConnection requirements, use connection::close instead of this function, as it also takes care of closing the underlying stream. The handler signature for this operation is void(boost::mysql::error_code).

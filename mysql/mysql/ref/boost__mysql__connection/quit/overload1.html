connection::quit (1 of 2 overloads)

Notifies the MySQL server that the client wants to end the session (sync with error code version).

Synopsis 
void
quit(
    error_code&,
    error_info&);
Description

Sends a quit request to the MySQL server. If the connection is using SSL, this function will also perform the SSL shutdown. You should close the underlying physical connection after calling this function. If the Stream template parameter fulfills the SocketConnection requirements, use connection::close instead of this function, as it also takes care of closing the underlying stream.

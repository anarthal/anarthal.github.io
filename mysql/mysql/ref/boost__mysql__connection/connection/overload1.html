connection::connection (1 of 2 overloads)

Initializing constructor.

Synopsis 
template<
    class... Args,
    class EnableIf = typename std::enable_if<std::is_constructible<Stream, Args...>::value>::type>
connection(
    Args&&... args);
Description

As part of the initialization, a Stream object is created by forwarding any passed in arguments to its constructor. The constructed connection will have connection::valid return true.

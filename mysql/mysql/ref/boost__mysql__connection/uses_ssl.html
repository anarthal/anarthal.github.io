connection::uses_ssl

Returns whether the connection uses SSL or not.

Synopsis 
bool
uses_ssl() const;
Description

This function always returns false if the underlying stream does not support SSL. This function always returns false for connections that haven't been established yet (handshake not run yet). If the handshake fails, the return value is undefined. This function can be used to determine whether you are using a SSL connection or not when using SSL negotiation (see this section).

connection::async_connect (2 of 2 overloads)

Performs a connection to the MySQL server (async with error_info version).

Synopsis

    typename EndpointType,

async_connect(
    const EndpointType& endpoint,

    error_info& output_info,
    CompletionToken&& token);
Description

This function is only available if Stream satisfies the SocketStream requirements. Connects the underlying stream and then performs the handshake with the server. The underlying stream is closed in case of error. Prefer this function to connection::async_handshake. The strings pointed to by params should be kept alive by the caller until the operation completes, as no copy is made by the library. If using a SSL-capable stream, the SSL handshake will be performed by this function. See this section for more info. The handler signature for this operation is void(boost::mysql::error_code).

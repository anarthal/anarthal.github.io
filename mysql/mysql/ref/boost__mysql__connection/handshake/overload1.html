connection::handshake (1 of 2 overloads)

Performs the MySQL-level handshake (sync with error code version).

Synopsis 
void
handshake(
    const connection_params& params,
    error_code& ec,
    error_info& info);
Description

Does not connect the underlying stream. If the Stream template parameter fulfills the SocketConnection requirements, use connection::connect instead of this function. If using a SSL-capable stream, the SSL handshake will be performed by this function. See this section for more info.

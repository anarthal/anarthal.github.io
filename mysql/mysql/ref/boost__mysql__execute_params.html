Description

In essence, this class contains an iterator range [first, last), pointing to a sequence of values that will be used as parameters to execute a prepared_statement. ValueForwardIterator must meet the ValueForwardIterator type requirements. In the future, this class may define extra members providing finer control over prepared statement execution. The make_execute_params helper functions make it easier to create instances of this class.
